# Frontend Dashboard

## Requirements
Build a React dashboard.

## Components

### Map View
- Show pipes (color-coded risk)
- Click to view details

### Alerts Panel
- List alerts with priority score

### Impact Simulator (IMPORTANT)
- Button: "Analyze Impact"
- Show:
  - Loss if ignored
  - Repair cost
  - Savings
  - Recommendation

### Graphs
- Flow & pressure trends

## Interaction
- Run simulation
- View alerts
- Trigger Impact Simulator
- Get AI explanation

## Data Flow
- Fetch alerts from API
- Use environment variables for API URL

## UI Design
- Clean, modern
- Blue/white theme
- Responsive

## Test
- Alerts display correctly
- Impact results shown
- API connected
